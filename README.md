# Daily Delights

A recipe website built as a project for the Web Technology Course.

## Description

Daily Delights is a simple, responsive recipe website that allows users to browse, search, and save their favorite recipes. The website features a clean interface with category filtering and a favorites system using local storage.

## Features

- **Browse Recipes**: View a collection of recipes displayed in a card-based grid layout
- **Search Functionality**: Search recipes by name using the search input
- **Category Filtering**: Filter recipes by categories (Breakfast, Lunch, Dinner, Snacks, Dessert)
- **Favorites System**: Save favorite recipes to local storage for quick access
- **Responsive Design**: Mobile-friendly layout using Bootstrap 5

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Bootstrap 5 (via CDN)

2. Open `index.html` in your web browser
